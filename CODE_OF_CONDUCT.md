# Code of Conduct

## Our Pledge

We want this project to be a welcoming, professional space for organizers, attendees, and contributors. We expect everyone participating in the project to treat others with respect.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful in discussions, reviews, and issues
- Giving constructive feedback and staying focused on the work
- Assuming good intent while still being clear about problems
- Respecting different experience levels and technical backgrounds

Examples of unacceptable behavior:

- Harassment, threats, or personal attacks
- Discriminatory, insulting, or demeaning language
- Publishing private information without permission
- Spam, trolling, or deliberately derailing discussions

## Scope

This Code of Conduct applies to project spaces, including issues, pull requests, discussions, and any other public interactions tied to this repository.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, code, issues, or other contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a private channel with the maintainer instead of escalating publicly. If no private contact is available, open a minimal issue requesting maintainer contact without posting sensitive details.
